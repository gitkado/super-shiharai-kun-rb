# Lessons Learned

Claudeが過去のセッションで犯したミスや学んだ教訓を蓄積するファイル。
セッション開始時にレビューし、同じミスを繰り返さないようにする。

## フォーマット

```markdown
### [日付] カテゴリ: 簡潔なタイトル

**問題**: 何が起きたか
**原因**: なぜ起きたか
**予防ルール**: 今後どうすべきか
```

## カテゴリ一覧

- `設計`: アーキテクチャ・設計判断のミス
- `実装`: コーディング・ロジックのミス
- `テスト`: テスト漏れ・不適切なテスト
- `ワークフロー`: 手順・プロセスの問題
- `コミュニケーション`: ユーザーとの認識齟齬
